# Baron Analytics v5.0

מערכת ניתוח מכירות לחנויות ומוצרים

## התקנה והרצה

```bash
npm install
npm run dev
```

האפליקציה תרוץ ב: http://localhost:3000

## מה חדש בגרסה 5.0

- ✅ מרחק מהשיא - מציג ערך עכשווי לצד השיא
- ✅ חזרות - מציג אחוז חזרות (לא כמות)
- ✅ ירידה רצופה - תוקן להציג רק חודשים רצופים
- ✅ מסך הגדרות - חדש
- ✅ מוצרים - עמודת מחזור מכירות

## סטטיסטיקות
- 1,075 חנויות (821 פעילות)
- 134 מוצרים
- נתונים: ינואר 2024 - דצמבר 2025
